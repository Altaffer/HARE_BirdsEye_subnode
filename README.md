Copy of the BirdsEye subscriber node for CRA programming sample.

BirdsEye's annotation system is a 2-stage process. The "sub_node" file emcpompasses the first step in which, the user parses the flight dataset from the ROS2-native bag file format into a sqlite database using this ROS2 node.
